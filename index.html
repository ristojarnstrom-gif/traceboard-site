Frequently asked questions.

Is TraceBoard Suite self-hosted only?

TraceBoard Suite is built for self-hosted deployment by default. We provide Docker and Kubernetes deployment options, plus detailed guides for common infrastructure setups. A managed cloud option is coming soon.

Can I use TraceBoard Suite without all three products?

Yes. You can use TraceBoard for requirements only, or skip TraceDocs if you prefer manual documents. However, the three products share the same data and users — running them together is where you get the most value.

How does pricing work?

TraceBoard Suite uses per-instance pricing, not per-user seats. The Community tier is free and open source. Professional and Premium tiers include advanced features like RBAC, audit logging, and API access. No surprise bills based on team size.

Is there vendor lock-in?

No. TraceBoard Suite is self-hosted on your infrastructure. You own your data. Export is straightforward — requirements, documents, and test results are all exportable.

Can I migrate from Excel or Jira?

Yes. We provide migration guides and tools for common sources like Jira and CSV/Excel imports. Contact support for detailed migration planning.

What support options are available?

Community tier gets GitHub Issues and community forums. Professional and Premium tiers include email support, with Premium including a dedicated technical account manager and SLA guarantees.
