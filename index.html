01 — How it works

Plan, Document, Prove.

Three products that share the same data, users, and authentication — deploy once, use all three.

Step 1 — Plan the work

TraceBoard

Create requirements, tasks, bugs, and tests. Link them together. Track progress on Kanban boards and create immutable baselines for compliance.

Step 2 — Document the work

TraceDocs

Create baselines, then generate SRS and Test Plan PDFs from your traceability data. Write and organize knowledge base articles with a Markdown editor and folders.

Step 3 — Prove the work

TraceTest

Execute test runs, record results (Pass/Fail/Blocked), and view coverage reports. Every result linked back to its requirement — full audit trail.

Questions

What happens if I stop renewing my self-hosted license?

The software keeps running with the features you already have. You stop receiving updates and support until you renew.

What is the difference between TraceBoard, TraceDocs, and TraceTest?

TraceBoard is the core traceability engine – manage requirements, tasks, bugs, tests, and trace links. TraceDocs generates compliance documents (SRS, Test Plans) from baselines and provides a Markdown knowledge base. TraceTest tracks test execution, records results, and shows coverage reports. All three share the same data, users, and authentication.

What does "Early Access" mean for TraceDocs and TraceTest?

TraceDocs and TraceTest are fully functional and ready for daily use. We are actively adding new features (custom templates, Word export, evidence management) based on customer feedback. By purchasing Professional or Team today, you lock in the current price and receive all future updates at no additional cost.

Can I move from Managed Cloud to self-hosted later?

Yes. Export your data and import it into your own instance. Same format either way, so there's no lock-in.

Is there a trial?

Yes — 30 days with the full current feature set

Do annual plans cost less than monthly?

Yes. Paying yearly on Managed Cloud costs the same as 10 months instead of 12. Self-hosted licenses are billed yearly.

Does TraceBoard Suite support air‑gapped deployments?

Yes. The self-hosted Docker Compose stack includes its own databases and runs fully offline. No external calls are made — your data stays entirely within your network.

What makes TraceBoard Suite different from Jira?

TraceBoard Suite is built specifically for traceability between requirements, tasks and tests — with one‑click baselines and compliance exports. It includes document generation (TraceDocs) and test execution (TraceTest) in the same platform. It's a single Docker Compose stack you own, not a cloud platform you subscribe to. All core traceability features are included in every plan.
